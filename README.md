# DVision
Open CV based Drone Navigation

## Getting Started
install ar-parrot drone node js api
install open-cv

### Prerequisites

run http-server

```
$ http-server
Starting up http-server, serving ./
Available on:
  http://127.0.0.1:8080
  http://192.168.1.2:8080
```

run base.js
